# Plantree Home 🌿

From plan to pantry.

Plantree Home is a meal planning and grocery coordination app designed for households.

It helps families:

- Plan meals for the week
- Manage shared grocery lists
- Track pantry ingredients
- See cooking activity and statistics

## Features (Work in Progress)

- Weekly meal planning
- Shared grocery lists
- Pantry ingredient tracking
- Cooking statistics
- Mobile-friendly dashboard

## Tech Stack

Angular  
TypeScript  
SCSS  
Standalone Components  

## Project Status

Work in progress — currently building the home dashboard and planning system.
